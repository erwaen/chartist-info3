Chartist.JS

Chartist is a simple responsive charting library built with SVG. There are hundreds of nice charting libraries already out there, but they are either:

  • use the wrong technologies for illustration (canvas)
  • weighs hundreds of kilobytes
  • are not flexible enough while keeping the configuration simple
  • are not friendly to designers
  • more annoying things

Objetivo

El objetivo de Chartist es proporcionar una biblioteca simple, liviana y no intrusiva para crear gráficos de manera responsiva en un sitio web.

Es importante comprender que una de las principales intenciones de Chartist es confiar en los estándares en lugar de proporcionar su propia solución a un problema que ya está resuelto por esos estándares.
Necesitamos aprovechar el poder de los navegadores hoy y despedirnos de la idea de resolver todos los problemas nosotros mismos.

Funcionamiento

Chartist trabaja con inline-SVG y tambien aprovecha el poder del DOM,
Esto también significa que Chartist no proporciona su propio manejo de eventos, etiquetas, comportamientos o cualquier otra cosa que ya se puede hacerse simplemente con HTML, JavaScript y CSS.

La única y única responsabilidad de Chartist es ayudarlo a dibujar "Gráficos simples responsivos" utilizando inline-SVG en el DOM, CSS para diseñar y JavaScript para proporcionar una API para configurar sus gráficos.

Estructura del codigo

La structura del codigo de Chartist es muy simple,
Chartis tiene 3 tipos de charts: Line, Bar y Pie,
Cada uno tiene su propia funcion pero las 3 reciben 'los mismos' 3 argumentos:

1. Target(id de un elemento del DOM)
2. Data
3. Options

Tambien se le puede agregar otro argumento mas para aplicar custom css

Entonces el codigo se veria de la siguiente manera:

new BarChart('#my_element_id', data, options );
new PieChart('#my_element_id2', data, options );
new LineChart('#my_element_id3', data, options );
